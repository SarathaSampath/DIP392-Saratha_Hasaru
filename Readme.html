Eazy Barber

Eazy Barber is a web-based salon management system that allows users to book appointments, manage time slots, and view their bookings. The project is designed to provide a seamless experience for both customers and administrators.

Features

User Features

  • Home Page: A welcoming interface with an option to book appointments.
  • Register: Users can sign up with their name, contact number, email, and password.
  • Login: Users can log in to access their personalized features.
  • Book Appointment: Users can view available time slots and book an appointment.
  • My Appointments: Users can view their booked appointments.
  • Contact Us: Users can view the contact details of the salon.

Admin Features

  • Admin Login: Admins can log in to access the admin panel.
  • Admin Panel: Admins can navigate to manage slots or view customer details.
  • Manage Slots: Admins can add new time slots and view all slots with their booking status and user details.
  • Customer Details: Admins can view customer information and their appointments.

Project Structure

Frontend

  • Pages:
    • index.php: The home page with a hero section and navigation links.

    • register.php: User registration page.

    • login.php: User login page.

    • book.php: Appointment booking page.

    • my_appointments.php: Displays the user's booked appointments.

    • contact.php: Contact information page.

    • admin_login.php: Admin login page.

    • admin_panel.php: Admin dashboard with navigation to manage slots and customer details.

    • manage_slots.php: Admin page to manage time slots.

    • customer_details.php: Admin page to view customer details.

  • Assets:
    • css/style.css: Custom styles for the project.
    • images/: Contains images like the logo, background, and favicon.

Backend

Database Connection

  • db.php: Handles the connection to the MySQL database.

API Endpoints

  • get_slots.php: Provides available slots for booking.
  • book_appointment.php: Handles appointment booking by checking slot availability and updating the database.
  • my_appointments.php: Returns the list of appointments for a logged-in user in JSON format.
  • login.php: Handles user login by verifying credentials and starting a session.
  • register.php: Handles user registration by storing user details in the database.

Admin Management

  • manage_slots.php: Allows admins to add new slots, delete slots (if not booked), and cancel appointments.
  • get_customer_data.php: Fetches customer details (ID, name, email, and contact) for admin viewing.

Database Schema

Tables


  • users
    • id: Primary key.
    • name: User's name.
    • contact: User's contact number.
    • email: User's email.
    • password: User's hashed password.

  • time_slots
    • id: Primary key.
    • slot_time: Time of the slot.
    • slot_date: Date of the slot.
    • is_booked: Boolean indicating if the slot is booked.

  • appointments
    • id: Primary key.
    • slot_id: Foreign key referencing time_slots.
    • user_id: Foreign key referencing users.
    • booking_time: Timestamp of the booking.

Installation

  1. Set up a local server (e.g., XAMPP or WAMP).
  2. Place the project in the htdocs directory.
  3. Import the database:
    • Open phpMyAdmin.
    • Import the provided SQL file (eazy_barber.sql).
  4. Update the database connection in backend/db.php: 
    $conn = new mysqli('localhost', 'root', '', 'eazy_barber');
  5. Start the server and access the project at http://localhost/eazy_baber_v2/frontend/index.php.

Usage

For Users

  1. Register on the platform.
  2. Log in using your credentials.
  3. Book an appointment by selecting an available time slot.
  4. View your bookings on the "My Appointments" page.

For Admins

  1. Log in to the admin panel via the "Admin Login" page.
  2. Navigate to "Manage Slots" to add or view time slots.
  3. Navigate to "Customer Details" to view customer information and appointments.
